Autokin

Build test automation with less or zero coding skills required. Leverage the power of Gherkin and write English-like scripts.

Get Started

Write automation scripts without coding.

Use Gherkin to write test automation and start running test in no time. Gherkin is easy to learn and understand as it is sound and written in English like syntax.

Automate Contract Test without the hassle.

Autokin supports REST API testing and allows you to write contract test to ensure that all consumer in your application expects the correct API.

Easily integrate to your continuous integration.

Autokin runner are in command line, which enables easy integration to your CI.

Getting Started with Autokin

Install autokin using NPM. After installation, create and initialised your project. Then off you go, run your test!
